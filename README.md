# Project Goudendraak

## How do i start with development?

First make sure you have the following installed:

```sh
    docker
    php version 8.2
```

```sh
    git clone git@github.com:RubenvanderHout/webfs-goudendraak.git
    cd webfs-goudendraak
    npm install
    composer install

    php la

```
